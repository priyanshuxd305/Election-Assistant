Election Assistant is a simple interactive chatbot-based web application that helps users understand the election process in an easy and structured way.

Features:
- Eligibility check (18+ voting rule)
- Voter registration guidance
- Step-by-step voting process
- Election timeline visualization
- Interactive chatbot interface

Tech Used:
- HTML
- CSS
- JavaScript

This project simplifies complex election procedures and improves awareness through an intuitive and user-friendly interface.
